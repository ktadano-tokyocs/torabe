# とらべLP 更新・運用ガイド

とらすとベース フリースクール（とらべ）ランディングページの**更新マニュアル**です。
非エンジニアの方でも、写真・お知らせ・説明会の情報を自力で更新できるようにまとめています。

---

## このサイトについて

- 本体は 1 枚の静的HTML（`index.html`）＋ Vue（CDN読み込み）で動いています。
- 特長セクションの写真は **WordPress メディア**（`tokyocs.org/wordpress`）から配信しています。
- お知らせ・説明会の情報は **Googleスプレッドシート**で管理し、自動でサイトに反映されます。

## 1. 特長セクションの写真を差し替える

「とらべが選ばれる4つの特長」の各写真は、WordPress メディアの**固定URL**で配信しています。
**同じファイル名のまま中身だけ上書き**すれば、HTMLを触らずにサイトへ反映されます。

### 現在の4枚

| 特長 | ファイル名 | 配信URL |
|---|---|---|
| 1. 日本語 | `feature1-japanese.jpg` | https://tokyocs.org/wordpress/wp-content/uploads/2026/06/feature1-japanese.jpg |
| 2. 探究 | `feature2-inquiry.jpg` | https://tokyocs.org/wordpress/wp-content/uploads/2026/06/feature2-inquiry.jpg |
| 3. 原体験 | `feature3-experience.jpg` | https://tokyocs.org/wordpress/wp-content/uploads/2026/06/feature3-experience.jpg |
| 4. 個別評価 | `feature4-evaluation.jpg` | https://tokyocs.org/wordpress/wp-content/uploads/2026/06/feature4-evaluation.jpg |

### 差し替え手順

1. WordPress管理画面にログインし、**メディア**を開く
2. 差し替えたい画像（例: `feature1-japanese.jpg`）を選択する
3. 編集画面の **「Enable Media Replace（メディアを置換）」** をクリック
4. **「ファイルを置換（ファイル名はそのまま）」** を選ぶ
   - ※ もう一方の「新しいファイル名にして更新」は**選ばないでください**（URLが変わってしまいます）
5. 新しい画像をアップロードして保存
6. サイトを開いて、写真が変わっていることを確認

これだけで完了です。**HTMLの編集は不要**です。

### 守っていただくルール

- ✅ **「ファイル名はそのまま」モード**を使う（URLを変えないため）
- ✅ **拡張子は `.jpg` のまま**差し替える（`.png` などに変えるとURLが変わります）
- ✅ 写真の**縦横比は 4:3 〜 16:9 程度**が崩れにくい（横長の写真が無難）
- ✅ サイズの目安は**横 1200px 前後**（大きすぎる画像は表示が重くなります）

### 反映されないとき

- WordPress 側のキャッシュ: 最新版の Enable Media Replace は主要キャッシュプラグインを自動でクリアします。
- ブラウザのキャッシュ: **スーパーリロード**（Mac: `Shift + ⌘ + R` / Windows: `Ctrl + F5`）で再読み込みしてください。

### （参考）Enable Media Replace が入っていない場合

WordPress管理画面 → **プラグイン → 新規追加** で「Enable Media Replace」を検索 → インストール → 有効化。
無料・公式プラグインです。

## 2. お知らせ・説明会を更新する

「お知らせ」と「説明会」は **Googleスプレッドシート**で管理しています。
スプレッドシートを編集して保存すると、Google Apps Script（GAS）経由で自動的にサイトへ反映されます（**サイト側の作業は不要**）。

### スプレッドシートの列構成

**「お知らせ」シート**

| 列 | 内容 | 例 |
|---|---|---|
| A | 日付 | 2026/06/16 |
| B | タイトル | 夏の体験会のお知らせ |
| C | 概要（任意） | 7月に体験会を開催します |
| D | リンクURL（任意・空欄可） | https://note.com/... |
| E | 公開 | `TRUE` = 表示 / `FALSE` = 非表示（下書き） |

**「説明会」シート**

| 列 | 内容 | 例 |
|---|---|---|
| A | 日付 | 2026/06/16 |
| B | 時間 | 19:00〜20:30 |
| C | 形式 | オンライン / 現地 |
| D | 公開 | `TRUE` = 表示 / `FALSE` = 非表示（下書き） |

### 更新手順

1. スプレッドシートに行を追加、または既存の行を編集する
2. 表示したい行は**公開列を `TRUE`** にする（下書きにしたい行は `FALSE`）
3. 保存すれば自動でサイトに反映されます

> お知らせは**新しい日付が上**に表示されます。

### 注意

- スプレッドシートの**データ更新だけなら、再デプロイは不要**です。
- GASのプログラム（`news.gs`）自体を変更した場合のみ、再デプロイが必要です（手順は `news.gs` 冒頭のコメント参照）。

## 3. 困ったとき（トラブルシュート）

**特長の写真が表示されない**
- URLが `https://` になっているか（`http://` だとブラウザにブロックされます）
- ファイル名・拡張子（`.jpg`）・月フォルダ（`2026/06`）が一致しているか
- その画像が WordPress メディアに存在するか

**お知らせ／説明会が表示されない**
- 公開列が `TRUE` になっているか
- スプレッドシートを保存したか（反映に少し時間がかかる場合があります）

- 画像URLの月フォルダ（`2026/06`）を変える必要が出た場合は、この4箇所を編集します。
- お知らせ・説明会を読み込む GAS のURL（`GAS_NEWS_URL`）は `index.html` の **1377行目付近**にあります。
